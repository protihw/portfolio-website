Factory Escape

Factory Escape é um jogo 3D singleplayer do genêro Escape Room. Foi um projeto desenvolvido para o Trabalho de Conclusão de Curso (TCC) do Curso de Jogos Digitais (SENAI-ES). Desenvolvido em C++ na engine Unity 3D por um grupo de 5 integrantes, dois deles desenvolvedores (eu e mais um).

Flappy Bird

Durante os estudos de Python recriei o clássico Flappy Bird utilizando Pygame Zero. Este projeto envolveu a programação do movimento do pássaro e dos obstáculos, assim como a implementação da lógica de pontuação. Foi uma oportunidade emocionante para aplicar meus conhecimentos em Python e explorar as possibilidades criativas oferecidas pelo Pygame Zero.

Severino Game

Desenvolvi um emocionante jogo de plataforma 2D na Unity, onde os jogadores assumem o papel de Severino, o personagem principal de "A Morte e a Vida Severina". Utilizando a linguagem C# e as ferramentas da Unity, criei um ambiente envolvente inspirado no sertão nordestino, repleto de desafios e obstáculos únicos. Habilidades em programação de jogos, design de níveis e familiaridade com a Unity foram essenciais para trazer essa jornada à vida.
